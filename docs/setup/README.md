- `odoo-installation.md` - Odoo Community installation guide

## Quick Start

1. Follow `environment-setup.md` to prepare your system
2. Install Odoo following `odoo-installation.md`
3. Configure social media APIs using `social-media-apis.md`
4. Run verification scripts to ensure everything works

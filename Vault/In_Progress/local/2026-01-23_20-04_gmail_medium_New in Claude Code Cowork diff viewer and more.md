Here's what's new


We've shipped several updates to Claude Code designed to help you build more, 
faster. From new ways to access Claude Code to the diff viewer on web and 
desktop, here are five new features and tools for a better Claude Code 
experience.



Cowork (research preview)

Cowork brings the power of Claude Code to non-coding tasks (no terminal 
required). Open the desktop app, connect your files and tools, and let Claude 
get things done. Available now for Max and Pro subscribers on macOS.

Download for macOS 
<https://claude.com/download?utm_source=it&utm_medium=email&utm_campaign=cc-newsletter&utm_term=january>



VS Code extension is generally available (GA)

Previously in beta, Claude Code for VS Code is now GA and is closer to the CLI 
experience: @-mention files for context, use familiar slash commands (/model, 
/mcp, /context), and more.

Install from the VS Code marketplace 
<https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code&utm_source=it&utm_medium=email&utm_campaign=cc-newsletter&utm_term=january>



Diff view on web and desktop

You can now see Claude Code's exact changes on web or desktop without leaving 
the app. View full diffs and leave inline comments to iterate with Claude, all 
in one place.

Try it on web 
<https://claude.ai/code/?utm_source=it&utm_medium=email&utm_campaign=cc-newsletter&utm_term=january>
 ordesktop 
<https://code.claude.com/docs/en/desktop?utm_source=it&utm_medium=email&utm_campaign=cc-newsletter&utm_term=january>



Claude Code in Slack

Delegate tasks and hand off context to Claude Code directly from Slack. You 
can now tag Claude in Slack to spin up a Claude Code session using context from 
recent messages.

Learn more

# Abdullah Junior Web Dashboard

This is the Next.js web dashboard for the Digital FTE system.

## Vercel Deployment

1.  Connect your repository to Vercel.
2.  Set the root directory to `frontend`.
3.  Add the environment variable `NEXT_PUBLIC_API_BASE_URL=https://your-fly-app.fly.dev`.
4.  Deploy!

## Local Development

```bash
npm install
npm run dev
```
